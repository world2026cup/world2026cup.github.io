# 2026 월드컵 Elo 시뮬레이션 웹 뷰어

`data/`의 팀·일정·결과 CSV를 읽어 시뮬레이션을 돌리고, 결과를 인터랙티브 웹페이지로 보여줍니다.

## 보기

`viewer/index.html`을 브라우저에서 더블클릭해서 열면 됩니다. (별도 서버 불필요 — 데이터는 `data.js`에 인라인되어 있음)

> 차트는 Chart.js를 CDN에서 불러오므로 처음 열 때 인터넷 연결이 필요합니다.

## 탭

- **순위 & 우승확률** — 우승 확률 순위표, 라운드별 진출 확률 막대, 우승 후보 시상대
- **일별 Elo 변화** — 진행된 경기에서 실제로 움직인 Elo (경기별 막대 + 일자별 로그)
- **우승확률 추이** — **경기 하나하나마다** 스냅샷으로 본 우승/결승/16강 확률 변화 + 최근(직전 경기) 상승·하락 팀 + **타임머신 슬라이더**(재생 버튼으로 경기별 진행 애니메이션)
- **다음 매치업** — 미진행 경기의 승·무·패 확률 (Poisson 모델) + 예상 득점
- **매치업 파인더** — 두 팀을 고르면 토너먼트에서 만날 확률을 라운드별로 표시
- **대진표** — 32강~결승 각 자리의 가장 유력한 팀(확률 막대) + 승자 강조, 팀 클릭 시 경로 강조
- **대륙별 분석** — 연맹별 팀 배정 vs 전력(Elo) vs 실제 우승확률 점유율 비교, 누적 Elo 변화

순위표에는 직전 스냅샷 대비 **순위 변동 화살표**(▲▼)가 표시됩니다.

> 모든 경기 시간은 **한국 시간(KST, UTC+9)**으로 표시됩니다. `build_data.py`가 각 경기의 개최지 `utc_offset`을 기준으로 변환합니다. (원본 일정 CSV의 C·D·G·I조 누락 오프셋은 실제 개최 도시를 확인해 채웠습니다: C·I조 동부 −4, D·G조 태평양 −7.)

## 팀별 페이지

순위표(또는 조 순위·예상 상대)에서 **팀 이름을 클릭**하면 그 팀 페이지(`#team/<영문명>`)로 이동합니다:

- 라운드별 진출 확률 (32강 → 우승)
- 32강 예상 상대 Top 5 (32강 진출 시 조건부 확률)
- **조별 순위 확률 (1·2·3·4위)**
- 소속 조 현재 순위 (해당 팀 강조)
- 경기 결과 & 향후 일정

## 조별 페이지

상단 **📊 조별 현황** 탭(또는 팀 페이지의 "전체 현황 보기")에서 조 페이지(`#group/<A~L>`)로 이동합니다:

- 현재 순위표(승점·득실·Elo)와 **조별 순위 확률(1·2·3·4위) 막대**
- 전체 일정 — 완료 경기 스코어 + **남은 경기의 승·무·패 확률**
- A~L 빠른 이동 셀렉터

> 32강 상대 분포·매치업 파인더·대진표 데이터는 `build_data.py`가 한 번의 추적 시뮬레이션으로 모든 토너먼트의 녹아웃 대진을 집계해 함께 생성합니다(전체 생성 시간 약 75초).

## 데이터 갱신

경기 결과가 추가되거나(`data/...played_results...csv`) 팀 Elo가 바뀌면 다시 생성:

```bash
PYTHONPATH=src python viewer/build_data.py
```

- 시뮬레이션 횟수/시드는 `build_data.py` 상단에서 조정:
  - `SIMULATIONS` (기본 100,000) — 현재 상태 권위 수치(순위표·팀 페이지·대진표·매치업)와 최종 스냅샷
  - `SNAPSHOT_SIMULATIONS` (기본 40,000) — 타임머신 중간 스냅샷(경기당 1개)
- 타임머신 스냅샷은 진행된 경기 **하나마다** 자동 생성됩니다 (개막 전 + N경기)
- 전체 생성 시간은 위 설정 기준 약 3~4분
